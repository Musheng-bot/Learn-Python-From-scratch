## 00 Introduction

### Python是什么？

python是一门`解释型`的`编程语言`。python脚本依赖一个名为`解释器`的程序运行，运行后可以根据`python的语法规则`进行一些操作。

### Python的用途

Python的编程语言相当广, 应用领域包括但不限于：

- 数据科学
- 人工智能
- 爬虫
- 测试
- 桌面组件
- 服务器后端
- 启动脚本

### Python的优缺点

优点:
1. 简单，好上手
2. 灵活
3. 调试方便
4. 生态丰富
5. 灵活的`胶水`语言

缺点:
1. 慢
2. 不强制声明类型导致的使用障碍
3. GIL带来多线程性能瓶颈
4. 对大型项目适应性有限

### 一些基础知识

- 计算机的硬件系统通常由五大部件构成，包括：**运算器**、**控制器**、**存储器**、**输入设备**和**输出设备**。其中，运算器和控制器放在一起就是我们常说的**中央处理器**（CPU），它的功能是执行各种运算和控制指令。
- 在计算机中一切都是以`二进制`形式存储数据的，也就是我们常说的01序列
- 计算机正在运行的一切都是存储在`内存`中，而一般的文件，文件夹都是存储在`硬盘`中
- python的运行都依赖`解释器`执行，而这个解释器本质上就是一个程序，它负责识别你写的代码，翻译成机器能看懂的样子给机器执行

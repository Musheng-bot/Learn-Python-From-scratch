### 万能的print函数

在我们刚刚接触python的时候就用过`print`，当时我们就用这个打印了`Hello World!`语句，在`python`中，我们就可以用它来打印几乎一切我们想要观测的事物。

```python
a = 1
print(a)
print(type(a))
print(a + 1)
# ...
```

`print`函数在python中是真的几乎万能的地位，凡是你能想到的基本都可以试试看用print打印出来，不管是python的标准库还是其他的库都会支持print打印功能。
